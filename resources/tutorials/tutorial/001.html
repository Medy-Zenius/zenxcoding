Step by step instruction to install Haskell:

  1. Install Haskell platform from This site
  2. Download Heat from This site
  3. Install haskell platform according to your operating system, and remember where you install them
  4. If you don't have java installed, install java
  5. Put your heat.jar in a friendly folder and create a shortcut for easy access to heat
  6. Double click the heat.jar file
  7. You will be asked at setup to connect heat with your haskell, click browse and find the haskell folder, once in there goto bin folder and select ghci.exe
  8. If you're not asked to connect haskell with heat, then choose from heat menu program-option and follow the step 7
  9. Successful installation will produce a prelude> prompt in heat where you can do things in haskell
  10. For mac users, highly recommended to update to yosemite or at least mavericks
  11. And then install JDK from this site
  12. When installing jdk, make sure you choose the right platoform (Mac)
  13. For mac users, the location of haskell platoform is usually at /usr/bin/ghci, needed when setting up HEAT
